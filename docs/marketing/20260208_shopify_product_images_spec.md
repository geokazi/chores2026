Specifications for creating product images and SEO metadata for the ChoreGami Shopify store. Messaging is derived from [blog.choregami.app](https://blog.choregami.app) content research targeting parent pain points.

## Copywriting Principles

### The Pain-First Rule

**First 5 words must make parents think "That's ME."**

| Weak (Feature-First) | Strong (Pain-First) |
|---------------------|---------------------|
| "Chore management system" | "Stop being the bad guy" |
| "Helps parents stop nagging" | "Exhausted from repeating yourself?" |
| "Points-based rewards" | "Kids actually WANT to help" |
| "Digital delivery" | "See results in 30 days" |
| "Family access" | "Real peace at home" |
| "Prepaid subscription" | "Less than a coffee" |

### Parent Pain Points (from blog research)

| Pain Point | Emotional Phrase |
|------------|------------------|
| Constant repetition | "I already asked you 10 times!" |
| Checking/nagging | "Did you do it yet?" on repeat |
| Mental load | "Brain with 37 tabs open" |
| Burnout | "Exhausted from being the bad guy" |
| Arguments | "Chore wars every single day" |
| Reminder fatigue | "Feeling like a broken record" |
| Invisible labor | "No one notices what I do" |

### Proof Points (from blog)

- **"95% reduction in chore-related arguments"** (within 30 days)
- **"Kids started helping without being asked"**
- **71% of household mental tasks fall on mothers**
- **57% of parents report burnout**

## Image Requirements

### Technical Specifications

| Property | Value |
|----------|-------|
| **Dimensions** | 1200 x 1200 px (square, Shopify standard) |
| **Format** | PNG (for transparency) or JPG (for photos) |
| **File size** | < 500KB per image |
| **Color mode** | sRGB |
| **Resolution** | 72 DPI (web optimized) |

### Recommended Image Count Per Product

- **Primary image**: Product card (shows in collection grid)
- **Secondary image**: App screenshot or feature highlight
- **Optional**: Lifestyle/family image

## Canva Template Setup

### Step 1: Create New Design
- Select "Custom size": 1200 x 1200 px
- Name: "ChoreGami Shopify - [Product Name]"

### Step 2: Background
- Add gradient using brand colors
- Keep it subtle (10-20% opacity shift)

### Step 3: Phone Mockup
- Search "iPhone mockup" in Elements
- Position center, ~60% of canvas height
- Insert app screenshot into frame

### Step 4: Badge
- Use "Shapes" â†’ Rounded rectangle or search "badge"
- Position top-right corner
- Add text with brand font (Inter or similar sans-serif)

### Step 5: Text Hierarchy
- **Product name**: 48-56px, Bold, Dark Green
- **Subhead**: 28-32px, Medium, Dark Green
- **Tagline**: 20-24px, Regular, Gray (`#6b7280`)
- **Price**: 36-44px, Bold, Primary Green

### Step 6: Export
- File â†’ Download â†’ PNG
- Quality: High
- Name: `choregami_[sku]_primary.png`
